# Prompts

This directory contains curated prompts for common development tasks.

## What are Prompts?

Prompts are carefully crafted instructions that help you get better results from AI tools. Good prompts are:

- Specific and clear
- Include relevant context
- Define expected output format
- Provide examples when helpful

## Categories

### Code Generation
Prompts for creating new code, functions, and classes

### Code Review
Prompts for analyzing and reviewing code quality

### Debugging
Prompts for identifying and fixing bugs

### Documentation
Prompts for generating documentation

### Refactoring
Prompts for improving code structure

### Testing
Prompts for creating and improving tests

## Using Prompts

1. Find a prompt that matches your task
2. Customize it with your specific details
3. Include relevant code or context
4. Review and iterate on the results

## Prompt Template Structure

Each prompt file includes:
- **Purpose**: What the prompt does
- **Template**: The prompt with placeholders
- **Example**: A filled-in example
- **Tips**: How to get better results

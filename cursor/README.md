# Cursor AI Integration

This directory contains examples and best practices for using AI within Cursor.

## Features

Cursor is an AI-first code editor with powerful features:

- **Ctrl/Cmd + K**: Inline code generation and editing
- **Ctrl/Cmd + L**: AI chat sidebar
- **AI Composer**: Multi-file editing capabilities
- **Code suggestions**: Advanced autocomplete with context awareness

## Setup

1. Download and install Cursor from [cursor.sh](https://cursor.sh)
2. Configure your AI model preferences
3. Set up your project rules and instructions

## Best Practices

1. **Use @-mentions**: Reference files, folders, docs, or code symbols in chat
2. **Define .cursorrules**: Create project-specific AI instructions
3. **Leverage Composer**: For multi-file changes and refactoring
4. **Use inline editing**: Press Ctrl/Cmd+K for quick edits
5. **Context selection**: Select relevant code before asking questions
